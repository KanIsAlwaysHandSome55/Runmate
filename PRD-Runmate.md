# PRD - Runmate

## 1. Overview
Runmate is a mobile app that allows users to track running distance, time, and simple progress.

## 2. Objectives
- Help users track running activity
- Show simple statistics
- Provide easy start/stop tracking

## 3. Target Users
- University students
- Beginner runners

## 4. Features
- Start run
- Stop run
- Track distance
- Track time
- View summary

## 5. Non-Functional Requirements
- Simple UI
- Fast performance
- Works offline
